Set Up My Blog

Oct 26th, 2014 12:00 am

I previously used my secondary Facebook account as a blog, but soon I felt need for another platform. I wanted to open my posting with my editor (emacs, org-mode) and update with my VCS (git). Simple static web site looked fine for me, as Discourse or Disqus seems to provide satisfiable commenting feature. I’ve looked through some examples below.

  • org web sites
  • org blogging tools
  • cse3521-website

As a total newbie on web front-end programming, org-mode, elisp, and blogging, I

  • myself doesn’t know exactly which feature I want.
  • am hard to deal with problems when I stuck.

Therefore, large and active community was especially important factor for me.

Octopress with org-octopress

I also noticed this post and tried it. I’m not sure of pros and cons of each, but I followed upper link because

  • It is in Elpa package list, while the other is not.

  • newer post of this blog suggests to use it, while the older post of this blog suggests to use the other.
